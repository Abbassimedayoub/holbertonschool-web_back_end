# Web Back End — Python & Node.js Track

## Overview

Holberton School back-end specialization track: **20 projects** covering modern Python (async, typing, testing), authentication, data persistence (MySQL advanced, NoSQL, Redis) and JavaScript back-end fundamentals (ES6, queuing systems, testing). Each directory is an independent project with its own README and tasks.

## Modules

### Python

| Module | Focus |
|---|---|
| `python_variable_annotations/` | Type annotations, duck typing |
| `python_async_function/` | `async`/`await`, `asyncio`, concurrent coroutines |
| `python_async_comprehension/` | Async generators and comprehensions |
| `pagination/` | REST pagination (page/size, hypermedia, deletion-resilient) |
| `caching/` | Caching policies: FIFO, LIFO, LRU, MRU |
| `personal_data/` | PII handling, log obfuscation, password hashing |
| `Basic_authentication/` | HTTP Basic auth from scratch on a Flask API |
| `Session_authentication/` | Session-based auth, cookies, session storage |
| `user_authentication_service/` | Complete auth service (register, login, password reset) |
| `i18n/` | Flask internationalization, locale and timezone selection |
| `Unittests_and_integration_tests/` | `unittest`, mocking, parameterization, integration tests |

### Databases

| Module | Focus |
|---|---|
| `MySQL_Advanced/` | Constraints, indexes, triggers, stored procedures, views |
| `NoSQL/` | MongoDB fundamentals, PyMongo queries and aggregation |
| `0x0B_redis_basic/` | Redis as a cache and store, call counting, history |

### JavaScript

| Module | Focus |
|---|---|
| `ES6_basic/` | let/const, arrow functions, spread, string templating |
| `ES6_classes/` | Classes, inheritance, metaprogramming |
| `ES6_data_manipulation/` | map/filter/reduce, typed arrays, Set/Map |
| `ES6_promise/` | Promises, async/await, error handling |
| `queuing_system_in_js/` | Redis operations, pub/sub, background job creation and processing |
| `unittests_in_js/` | Mocha, Chai, Sinon (spies, stubs), integration tests |

## Technologies

Python 3 (Flask, PyMongo), MySQL, MongoDB, Redis, Node.js (ES6+), Mocha/Chai/Sinon.

## Skills Demonstrated

Asynchronous programming, API authentication (Basic, session, token), data-layer design across SQL/NoSQL/cache, internationalization, and test-driven back-end development in both Python and JavaScript.
